
# Interactions entre l’homme et la machine sur le *Web*

L'**interface homme-machine (IHM) sur le Web** désigne l'ensemble des éléments qui permettent à un utilisateur d'interagir avec une application ou un site web. Cela comprend :

- Les éléments visuels : boutons, formulaires, menus, images, textes cliquables…
- Les interactions : clics, frappes au clavier, glisser-déposer, survol de la souris…
- Les retours visuels : animations, messages d'erreur, changements de couleur…

Ces **interactions** entre l’homme et la machine sur le *Web* reposent sur la gestion d'**événements** : quand l'utilisateur effectue une action (ex. cliquer sur un bouton), cela déclenche un événement qui exécute du code (souvent en JavaScript ou PHP) pour produire une réponse (afficher un message, envoyer des données, changer la page…).
